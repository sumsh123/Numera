# Numera — Scientific Learning Calculator

A scientific calculator built from scratch in C using Raylib, designed to go beyond calculations by combining computation, visualization, and mathematical learning.

Numera is a graphical calculator that supports basic arithmetic, scientific functions, function plotting, and an integrated information system that explains mathematical concepts, their properties, history, and real-world applications.

## Features

### Scientific Calculator
- Addition, subtraction, multiplication, and division
- Trigonometric functions:
  - sin(x)
  - cos(x)
  - tan(x)
- Logarithmic functions:
  - log(x)
  - ln(x)
- Square root calculations
- Square function (x²)

### Graphing Mode
- Visual representation of mathematical functions
- Supports:
  - Sine waves
  - Cosine waves
  - Tangent functions
  - Logarithmic functions
  - Natural logarithms
  - Square root functions

### Mathematical Learning System
Instead of only displaying answers, Numera provides information about:
- Numbers and their properties
- Mathematical operators
- Scientific functions
- Historical background of mathematical concepts
- Practical applications

### Interactive GUI
- Custom interface created completely with Raylib
- Custom fonts and color themes
- Button hover effects
- Click sound feedback
- Dedicated information popups

## Technologies Used

- C Programming Language
- Raylib Graphics Library
- Math.h mathematical functions
- Custom GUI rendering
- Audio integration
- Font loading and asset management

## Concepts Applied

This project explores important software engineering concepts including:

- Low-level programming in C
- Memory and string management
- Event-driven programming
- GUI development without external frameworks
- Mathematical computation
- Data visualization
- User interface design
- Modular project organization

## Motivation

Most calculators are designed only to produce results. Numera was built with a different approach: making mathematics more interactive and understandable.

The goal was to create a calculator that allows users to not only solve problems, but also explore the ideas behind the calculations through graphs, explanations, and mathematical history.

This project represents an exploration of programming fundamentals, computer graphics, mathematics, and building software from the ground up.

## Future Improvements
